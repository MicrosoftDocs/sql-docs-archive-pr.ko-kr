# <a name="sql-server-clr-object"></a>SQL Server, CLR 개체

 다음 테이블에서는 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] **CLR** 카운터에 대해 설명합니다.  
  
|CLR 카운터|Description|  
|------------------|-----------------|  
|**CLR Execution**|CLR에서 총 실행 시간(마이크로초)|
