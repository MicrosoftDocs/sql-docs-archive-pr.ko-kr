 <span data-ttu-id="b7301-156">관리 코드를 사용하면 메모리, 스레드 및 동기화와 같은 리소스 사용을 [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)]가 관리합니다.</span><span class="sxs-lookup"><span
